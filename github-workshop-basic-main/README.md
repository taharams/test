# workshop

## 事前準備（CodeSpaces）

<span style="font-size:1em; color: red; ">GitHubにサインイン済みの場合は、2～4の作業は不要です。</span>

1. ブラウザで`https://github.com/`にアクセス

2. `Sign in`をクリック

   ![Top](img/Top.png)

3. メールアドレスとパスワードを入力して、`Sign in`をクリック

   ![Sign in to GitHub](img/Sign%20in%20to%20GitHub.png)

<div style="page-break-before:always"></div>

4. 二要素認証のコードを入力して、`Verify`をクリック

   ![Sign in to GitHub](img/Two-factor%20authentication.png)

5. `https://github.com/払出Organization名/github-workshop-basic`にアクセス
- 上記URLに含まれる払出Organization名は適宜変更してください。

<span style="font-size:1em; color: red; ">上記URLのアクセスで、404エラーになる場合は、払出Organizationに参加できていないことが原因です。払出Organizationへの招待メールのJoin @払出Organization名のリンクをクリックして払出Organizationに参加した後、改めて上記URLのアクセスを試してください。</span>

6. `<> Code`をクリック

   ![Sign in to GitHub](img/Workshop-Materials.png)

<div style="page-break-before:always"></div>

7. `Codespaces`をクリック

   ![Sign in to GitHub](img/Code%20menu.png)

8. `+`をクリック

   ![Sign in to GitHub](img/Create%20a%20codespace.png)

<div style="page-break-before:always"></div>
9. 別タブでCodeSpacesのウィンドウが開く

   ![Sign in to GitHub](img/Setting%20codespace.png)

10. 10分程度待機する（ウィンドウの右下に表示されるポップアップは無視してよい）

11. `java Ready`の表示の有無を確認する
   ![Sign in to GitHub](img/Check%20codespace.png)

<div style="page-break-before:always"></div>
<span style="font-size:1em; color: red; ">java Readyの表示がある場合は、12～14の作業は不要です。</span>

12. エクスプローラーを展開して、Worshop1のApp.javaを開く
   ![Sign in to GitHub](img/OpenJava%20codespace.png)

13. `java Ready`の表示が出るまで待機する（2～3分程度）
   ![Sign in to GitHub](img/ActivateJava%20codespace.png)

<div style="page-break-before:always"></div>

14. `java Ready`の表示を確認する
   ![Sign in to GitHub](img/JavaReady%20codespace.png)

15. CodeSpacesのウィンドウを`×`をクリックして閉じる
   ![Sign in to GitHub](img/Close%20codespace.png)
